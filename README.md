# RTI Request Generator & Response Quality Analyzer

## 📌 Description
This project simplifies RTI filing using a guided generator and evaluates government responses using software engineering quality metrics.

## 🚀 Features
- RTI application generator
- Authority suggestion API
- Response Quality Analyzer
- Score based on:
  - Response Time (Latency)
  - Completeness (Coverage)
  - Compliance (Spec Conformance)

## 🧠 Novel Contribution
Treats RTI requests as test cases and government responses as system outputs.

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js (Express)

## ▶️ Steps to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm run dev
